# niri-dotfiles
**My niri + noctalia shell configuration**

![Screenshot](https://github.com/L1aRd/niri-dotfiles/blob/assets/Screenshot.png)

## Dependencies

- niri
- noctalia
- kitty
- fish
- nautilus
- neovim
- Bibata-Modern-Ice

## Installation

```
git clone https://github.com/L1aRd/niri-dotfiles.git
```
```
cd niri-dotfiles
```
```
cp -rpi fish kitty niri noctalia nvim ~/.config/
```
